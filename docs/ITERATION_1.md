# Iteration 1

## Overview

Iteration 1 focused on implementing the core features of the Hospital Appointment System. 
This iteration aimed to deliver the minimum viable product (MVP) including authentication, 
doctor management, and appointment booking.

## Iteration 1 Goals

- User Authentication
- Doctor Management
- Appointment Booking
- View Appointment Details
- Frontend Setup

## User Stories Completed

- Frontend development
- Account creation
- Login & security
- Doctor profile CRUD
- Manage doctor schedules
- View doctors & availability
- Book appointment
- View appointment details
